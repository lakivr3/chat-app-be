# Chat App - Backend

This is the backend of a real-time chat application, built with Node.js, Express.js, and Socket.io for real-time messaging. It includes secure authentication with JWT and stores messages persistently using MongoDB.

## Features

- Real-time messaging with WebSockets (Socket.io)
- Secure authentication using JWT
- Scalable architecture with Node.js and Express.js
- Persistent message storage using MongoDB
- RESTful API for user authentication and chat interactions

## Tech Stack

- **Node.js** - Backend runtime
- **Express.js** - Web framework for Node.js
- **Socket.io** - WebSocket communication
- **MongoDB** - Database for storing messages and user data
- **JWT** - Secure authentication mechanism
- **dotenv** - Environment variable management
